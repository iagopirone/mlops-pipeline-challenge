# Atribuição dos dados

Este kit usa o **BCCD Dataset** (Blood Cell Count and Detection).

- Origem: https://github.com/Shenggan/BCCD_Dataset
- Licença: MIT
- Modificações neste kit: anotações convertidas de Pascal VOC (XML) para o formato
  YOLO (`.txt` normalizado) e particionadas em `data/raw` (seed), `data/stream`
  (imagens) e `data/oracle` (labels do stream).

O BCCD é distribuído sob a licença MIT; consulte o repositório de origem para o
texto completo da licença e os créditos aos autores.
